== Description ==

Dokme Plus is a lightweight WordPress plugin that lets you create custom buttons and display them anywhere on your site using a shortcode.

**Key features:**
- Create up to 3 buttons without a license key
- Add unlimited buttons with a valid license key
- Multiple button actions:
  * Open a link
  * Copy text to clipboard
  * Share text (using Web Share API)
  * Call a phone number
  * Send an SMS message
- Customize button color, text, and font size
- Easy-to-use admin interface
- Supports Persian (fa) and English (en)

**Shortcode Example:**
```html
[dokmeplus id="123456789"]
